Definition: Matrix Entrywise Addition

Let \(\mathcal{M}_S(m, n)\) be an \(m \times n\) matrix space over \(S\) over an algebraic structure \(\left({S, \circ}\right)\). Let \(A, B \in \mathcal{M}_S(m, n)\). The matrix entrywise sum of \(A\) and \(B\), written \(A + B\), is defined in the following manner.

Let \(A + B = C = \left[{c}\right]_{mn}\).$$\forall i \in [1 \,.\,.\, m], j \in [1 \,.\,.\, n] : c(i, j) = a(i, j) \circ b(i, j).$$\(\left[{c}\right]_{mn}\) is the resulting \(m \times n\) matrix formed from performing these operations. The matrix entrywise sum is only defined if A and B have the the same dimensions.
